Returns or sets the command prefix.<br />
syntax: `!prefix (prefix|none|default)`<br />
<br />
If no prefix is provided, the current prefix will be shown.<br />
If the prefix is "default", the prefix will be reset to the bot's default prefix.<br />
If the prefix is "none", the prefix will be removed entirely, only allowing mentions to run commands.<br />
<br />
ADMINISTRATOR permission required (to change the prefix).
